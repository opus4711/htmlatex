Detailed Description

This class stores settings variables for the application in a binary file.

The settings are stored in a key-value-pair data structure (an associative array). The class member DEBUG is used all over the application to provide additional information about the steps of the processings if desired (verbose). Saving to and loading from the binary file is performed "automatically". When a variable is requested or set the loading or saving is performed implicitly.
