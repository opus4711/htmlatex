Detailed Description

This class describes how a reference to a subdocument of an input document is defined.

Example: In (X)HTML a subdocumet is referenced using the tag with its attribute "href". The TranslationMapper creates an instance of the DocumentReaderData class using the document definition file. Finally the DocumentReader obtains the DocumentReaderData object from the TranslationMapper.
