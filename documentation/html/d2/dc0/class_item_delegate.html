Detailed Description

This class is derived from the QItemDelegate class.

In this application its purpose is to control how the mainWindow's treeView displays data. The paint() method influences the drawing of the underlying widget directly. For further information look up QItemDelegate and QTreeView in the Qt documentation.

void ItemDelegate::paint ( QPainter *  painter,
const QStyleOptionViewItem &  option,
const QModelIndex &  index 
) const

This method draws the data on the underlying mainWindow's treeView.

Parameters:
<painter> is a pointer which allows drawing on the mainWindow's treeView.
Parameters:
<option> holds various kinds of information such as the coordinates and size of the rectangle on which the node data is supposed to be drawn. It also provides information whether or not the currently drawn node is selected. Notice that this method is called for each node of the mainWindow's treeView every time the treeView is repainted what can be caused be various events such as resizing the widget.
Parameters:
<index> is the index of the currently drawn node. Using the underlying model data structure the node can be identified and accessed by means of the given index.
