CDocumentData::CDocumentData ( QFileInfo  fileinfo,
CNode *  node,
FileType  filetype 
)

This class holds information of one document (i.e.

webpage). The information consists of the file path (URL), a reference pointing to the corresponding node in the document tree, which represents the whole document. This also stores a QFileInfo-object for the whole documents' index document. The appropriate document preprocessing is chosen by means of distinguished file types.
